# Self-Care Skill — Evaluation Criteria

## Daily Morning Routine
1. Memory health was checked before starting work? (yes/no)
2. Disk usage was checked? (yes/no)
3. Credential validity was verified? (yes/no)
4. Git state is clean (no uncommitted work from yesterday)? (yes/no)

**Pass threshold:** 4/4 — all checks must run

## Weekly Maintenance
1. Skill audit was completed with success rates listed? (yes/no)
2. Landscape watch was conducted and findings saved? (yes/no)
3. Stale memory/research files were flagged? (yes/no)
4. Findings were reported to human (or confirmed nothing to report)? (yes/no)

**Pass threshold:** 4/4

## Monthly Review
1. Anthropic release notes were read in full? (yes/no)
2. claude-desktop-debian version was checked against latest? (yes/no)
3. Performance report was generated and sent? (yes/no)
4. At least one actionable recommendation was made? (yes/no)

**Pass threshold:** 3/4

## Error Recovery
1. Crash cause was identified? (yes/no)
2. Workspace was left in clean state? (yes/no)
3. Human was notified with clear status? (yes/no)

**Pass threshold:** 3/3 — all must pass for a valid recovery
